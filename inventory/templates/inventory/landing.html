Project Mission

Avis AI acts as a cohesive decision-support architecture, built specifically to eliminate high latency in retail inventory operations and secure transit audits.

By integrating machine learning algorithms (Katana AI) directly with real-time logistical constraints (weather trackers) and sequential cryptography (SHA-256 blocks), Avis AI bridges the gap between complex analytical systems and daily supply operations.

Operational Workflow

1. Order Placement

Store Manager submits a purchase request with specific quantity targets.

2. Logistics Shield

The API radar monitors weather parameters to flag delivery alerts.

3. Vendor Fulfillment

Supplier logs intake details and submits confirmation blocks on completion.

4. Ledger Sealing

Orders are sequentially block-hashed to create an immutable audit record.

Core Subsystem Features

Katana AI Forecasting

Uses scikit-learn Linear Regression models to evaluate sales history on Pandas dataframes, predicting seasonal demand trends and automatically adjusting optimal safety stock thresholds. Runs asynchronously in background threads.

Logistics Delay Radar

Connects to the OpenWeatherMap API to fetch current atmospheric states in supplier operating cities. Flags active orders automatically with heavy rain, snow, or storm warnings on the manager's panel.

Cryptographic Audits

Constructs a SHA-256 block-linked hash chain for every finalized transaction. If any details are altered directly in the database, the integrity validator flags a mismatched signature immediately.
